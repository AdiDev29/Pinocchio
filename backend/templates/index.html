Get Started

  1. Download the Extension - Click the button below to download the Pinocchio extension zip file to your computer.
  2. Unzip the File - Locate the downloaded file in your Downloads folder and extract/unzip it to a location you'll remember.
  3. Open Chrome Extensions Page - Type chrome://extensions in your Chrome address bar and press Enter.
  4. Enable Developer Mode - Toggle on "Developer mode" in the top-right corner of the extensions page.
  5. Load the Extension - Click "Load unpacked" and select the folder you extracted in step 2.
  6. Start Using Pinocchio - The Pinocchio icon should now appear in your browser toolbar. Click it on any web page to analyze content.
